#### setRedstone()
```
mollusk.setRedstone([value [, duration]])
```
Sets the redstone signal strength on all sides simultaneously. If `duration` is specified, the signal will automatically turn off after that many seconds.
##### Parameters
1. value=0 : number The signal strength (0-15)
2. duration=nil : number The time in seconds to keep the signal on (nil = indefinitely)
##### Returns
1. boolean Whether the operation completed successfully

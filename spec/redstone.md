---

#### setRedstone()
```
mollusk.setRedstone([value])
```
Sets the redstone signal strength on all sides simultaneously.
##### Parameters
1. value=0 : number The signal strength (0-15)
##### Returns
1. boolean Whether the operation completed successfully
